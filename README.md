# Ugandan Flag – Python Graphics Project

This project was created as part of **Stanford University’s Code in Place 2026** course.  
It uses the `graphics` library to draw the **Ugandan National Flag** on a canvas, including:

- Six horizontal stripes (black, yellow, red)
- A centered white oval
- A crested crane emblem placed in the middle

## 🖥️ Technologies Used
- Python
- Code in Place `graphics` library
- Canvas-based drawing

## 📌 Project Description
The program opens a graphical window and draws the Ugandan flag using simple shapes such as rectangles and ovals.  
It also loads an external image of the **crested crane**, the national symbol of Uganda.

This project helped me practice:
- Coordinate systems  
- Drawing shapes programmatically  
- Working with images  
- Structuring a Python program with functions  

## ▶️ How to Run
This project runs in the **Code in Place environment**, which includes the `graphics` library.

To run it:
1. Copy the code into a Python file (e.g., `draw_flag.py`)
2. Run it inside the Code in Place IDE or any environment that supports the CIP `graphics` module.
